web.xml配置文件详解

一、web.xml是什么？

首先web.xml是java web项目的一个重要的配置文件，但是web.xml文件并不是Java Web工程必须的。

web.xml文件是用来配置：欢迎页、servlet、filer等的。当你的web工程没用到这些时，可以不需要使用web.xml文件来配置你的web工程。
